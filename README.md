# imdb-movie-review-sentiment-analysis

## Overview
This project focuses on classifying IMDb movie reviews as **positive** or **negative** using NLP and machine learning techniques.  
It demonstrates a complete text processing workflow, including data cleaning, feature extraction (TF-IDF), and model evaluation.

## Objectives
- Perform sentiment analysis on text data  
- Preprocess text (tokenization, stopword removal, lemmatization)  
- Apply ML models: Logistic Regression, Naive Bayes, Random Forest, Linear SVM  
- Visualize data and compare model performance

## Data Preprocessing
- Converted text to lowercase  
- Removed punctuation and HTML tags  
- Tokenized and lemmatized words  
- Used TF-IDF to convert text to numeric features

## Models Implemented
| Model | Accuracy | F1-Score |
|--------|-----------|----------|
| Logistic Regression | ~90% | 0.89 |
| Linear SVM | ~91% | 0.90 |
| Naive Bayes | ~87% | 0.86 |
| Random Forest | ~85% | 0.84 |

## Visualizations
- Sentiment distribution  
- Review length distribution  
- WordClouds (positive vs negative)  
- Confusion matrices and model comparison bar charts  

## Technologies Used
Python, Pandas, NumPy, scikit-learn, Matplotlib, Seaborn, NLTK
